# Colégio Curioso

Projeto Front-End autoral desenvolvido para a unidade Start by Alura.

## Sobre o projeto

O **Colégio Curioso** é um site criativo sobre o ambiente escolar, criado para transformar elementos comuns do colégio em ideias visuais, narrativas e interativas.

A proposta do projeto é unir identidade visual, acessibilidade, organização de código e criatividade. O site possui uma estética inspirada em pixel art e conta com uma interação em JavaScript: um gerador de missões criativas.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

## Funcionalidades

- Layout responsivo para diferentes tamanhos de tela.
- Navegação por seções.
- Pixel art autoral criada com HTML e CSS.
- Gerador interativo de missões criativas usando JavaScript.
- Estrutura semântica com foco em acessibilidade.

## Acessibilidade

O projeto utiliza boas práticas como:

- HTML semântico;
- textos com bom contraste;
- links e botões acessíveis por teclado;
- uso de `aria-label` e `aria-live`;
- layout responsivo para celulares, tablets e computadores.

## Identidade visual

A identidade visual utiliza uma paleta com tons de azul, verde e amarelo, criando uma aparência curiosa, escolar e tecnológica. A pixel art foi usada como elemento autoral para reforçar a personalidade do site.

## Como acessar

1. Baixe os arquivos do projeto.
2. Abra o arquivo `index.html` no navegador.
3. Clique no botão "Gerar missão" para testar a interação.
